# cats-vs-dogs-classification
This project detects cat and dog from image using transfer learning.

dataset link: https://www.kaggle.com/competitions/dogs-vs-cats/data

Place extracted files in:
- `data/train/cats`
- `data/train/dogs`
- `data/test/cats`
- `data/test/dogs`

## Setup
```bash
pip install -r requirements.txt
```

## Project Structure
- `src/` - contains training and model code
- `app/` - contains streamlit app code
- `models/` - contains saved models
